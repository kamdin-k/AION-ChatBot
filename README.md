# AION ChatBot

AION ChatBot is an AI-powered web application that lets users send prompts through a simple web interface and receive contextual responses generated by a Large Language Model API.

The project combines a modern frontend with a lightweight backend to create a smooth chatbot experience. It is designed to demonstrate AI integration, API communication, structured prompt handling, and full-stack web development.

## Features

- AI chatbot interface for user interaction
- Contextual response generation using LLM APIs
- Frontend and backend separation for clean architecture
- REST API communication between client and server
- JSON-based request and response handling
- Environment variable support for secure API key management
- Docker support for easier setup and deployment
